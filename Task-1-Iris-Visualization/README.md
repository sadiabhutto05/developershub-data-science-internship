# Task 1: Exploring and Visualizing Iris Dataset

## Introduction
This project is part of the Data Science & Analytics Internship tasks. The main purpose of this task is to understand the basics of data analysis and data visualization using Python. The Iris dataset was used to explore different flower species and analyze relationships between their features.

---

## Objective
The objective of this project is to:
- Load and inspect the Iris dataset
- Understand the structure of the dataset
- Perform basic exploratory data analysis (EDA)
- Visualize relationships and distributions of variables
- Identify patterns, variations, and possible outliers in the data

---

## Dataset Description
The Iris dataset is one of the most popular datasets in machine learning and data science. It contains measurements of iris flowers from three different species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

### Features in Dataset
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

The dataset helps in understanding how flower characteristics differ among species.

---

## Technologies and Libraries Used
The following Python libraries were used in this project:

- **Pandas**
  - For loading and analyzing the dataset

- **Matplotlib**
  - For creating basic data visualizations

- **Seaborn**
  - For advanced and attractive statistical plots

---

## Steps Performed

### 1. Data Loading
The dataset was loaded from a CSV file using Pandas.

### 2. Data Inspection
Basic dataset exploration was performed using:
- `.shape`
- `.columns`
- `.head()`
- `.info()`
- `.describe()`

These functions helped in understanding:
- Number of rows and columns
- Feature names
- Data types
- Statistical summary

### 3. Data Visualization
Different visualizations were created to better understand the dataset.

#### Scatter Plot
A scatter plot was created to analyze relationships between sepal length and petal length for different flower species.

#### Histogram
A histogram was used to observe the distribution of sepal length values.

#### Box Plot
A box plot was created to identify:
- Spread of data
- Median values
- Possible outliers
- Comparison between species

---

## Key Insights
- Different iris species have clearly different petal and sepal measurements.
- Petal length is very useful for distinguishing flower species.
- Histograms helped understand feature distribution.
- Box plots highlighted variations and possible outliers in the dataset.

The task improved understanding of how visual analysis helps in extracting meaningful insights from datasets using Python.
