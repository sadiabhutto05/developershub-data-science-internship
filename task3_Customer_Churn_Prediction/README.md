# Task 3 - Customer Churn Prediction

## Objective
Predict whether a bank customer will leave the bank.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## Model
Random Forest Classifier

## Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report

## Key Insights
- Age and balance highly affect churn.
- Customers with fewer products are more likely to leave.
